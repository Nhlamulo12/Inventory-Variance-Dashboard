# Inventory Variance Analysis Dashboard

This project analyzes warehouse inventory variance to identify high-risk SKUs and prioritize cycle counts.

## Problem
Manual cycle counting of all 1000+ SKUs was time consuming. No visibility on which parts had biggest variance.

## Solution
Used Python, Pandas, SQL, and Excel to:
1. Import cycle count data and compare `system_qty` vs `physical_qty`
2. Calculate variance % and flag "URGENT COUNT" items
3. Generate weekly Priority Count List in Excel
4. Visualize top 10 variance items using Matplotlib/Seaborn

## Tech Stack
- Python, Pandas, SQLite, SQL
- Excel, Data Visualization
- Jupyter Notebook

## Results
Reduced manual analysis time and proposed data-driven cycle count approach.
